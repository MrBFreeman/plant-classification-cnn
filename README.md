Автоматическая классификация сортов растений по изображениям семян или листьев
1. Описание проекта

Данный проект посвящён задаче автоматической классификации сортов растений по изображениям семян с использованием методов глубокого обучения и компьютерного зрения.

Задача формулируется как мультиклассовая классификация изображений, где каждому изображению соответствует один из 12 классов растений. В качестве модели используется сверточная нейронная сеть с переносом обучения (transfer learning).

Проект реализован на Python с использованием библиотеки PyTorch.

2. Цель и мотивация

Основная цель проекта — разработка модели глубокого обучения, способной автоматически распознавать виды растений по визуальным признакам семян с высокой точностью.

Практическая значимость:

автоматизация аграрных и биологических исследований

снижение ошибок ручной классификации

применение компьютерного зрения в сельском хозяйстве

повышение скорости анализа больших массивов данных изображений

4. Архитектура модели

В качестве базовой модели используется:

ResNet50, предобученная на датасете ImageNet.

Основные изменения:

удалён оригинальный классификационный слой

добавлен новый полносвязный слой размерности:

2048 → number_of_classes

Причины выбора ResNet50:

высокая устойчивость к переобучению

глубокая архитектура с residual connections

хорошие результаты на задачах компьютерного зрения

эффективная работа при переносе обучения

5. Предобработка и аугментация данных

Для повышения обобщающей способности модели применялись следующие методы:

изменение размера изображений до 224×224

случайное горизонтальное отражение

случайный поворот изображений

изменение яркости и контрастности

нормализация значений пикселей

Используемые параметры соответствуют стандартам ImageNet.

6. Параметры обучения

Основные гиперпараметры:

Параметр	Значение
Batch size	32
Learning rate	0.001
Optimizer	Adam
Loss function	CrossEntropyLoss
Epochs	10
Base model	ResNet50

Обучение проводилось с замороженными весами базовой сети и обучением только классификационного слоя.
